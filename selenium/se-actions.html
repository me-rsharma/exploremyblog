Let's explore mouse & keyboard operations using Selenium & Python

By Rajeshwar Sharma

Imagine a situation where you are lying on the floor and there is a chair on which you have kept your laptop and enjoying one of your Netflix series.

Suddenly you got a call from your manager asking to join a meeting invite where you need to enter the meeting id and password. But you know what, you have to enter the credentials by hovering over the meeting id and password field and unfortunately your laptop touchpad is not working and you are so lazy that you don't want to get the mouse which is kept probably in another room next to you and you have 15 mins in your hands.

In this situation, I would prefer suggesting you to write automation code and navigate to the link and do the necessary mouse and keyboards operations that need to be done.

You: Wait! do you really think this can be achieved in 15 mins?

Me: Yes, Selenium can help you with that.

What I feel, Selenium would have really thought about how lazy the people gonna be in this age and hence he has already developed an API that helps to simulate mouse & keyboard operations.

Selenium WebDriver's Advanced User Interactions API allows us to perform operations from keyboard events and simple mouse events to complex events such as dragging-and-dropping, holding a key and then performing mouse operations using the ActionChains class and building a complex chain of events exactly like a user doing these manually.

In this article, we will be exploring the different mouse & keyboard actions that can be performed with web driver API. So let's get dive into Selenium Webdriver's ActionChains class

Setting up ActionChains in automation script

  • Firstly, we need to import the package of ActionChains class as follows:

    from selenium.webdriver.common.action_chains import ActionChains

  • Create an object of ActionChains

    actions = ActionChains(driver)

Now. we are good to go and perform any mouse & keyboard operations using ActionChains

Usage of ActionChains class in automaton script

  • Mouse-hovering on element
  • Drag and drop the element on a webpage
  • Performing right-click operation
  • Pressing keyboard key

    • üëâNOTE: As per the Selenium documentation, key_down & key_up are only recommended with modifier keys (Control, Alt and Shift)

    For more info, please refer: https://www.selenium.dev/selenium/docs/api/py/webdriver/selenium.webdriver.common.action_chains.html#selenium.webdriver.common.action_chains.ActionChains.key_down

Usage of ActionChains class in automaton script for complex interactions

One of the great feature of Advanced User Interactions API is chaining the actions to perform complex operation on website. Even, the class name ActionChains itself suggesting chaining actions.

Let's perform some complex operations by chaining different actions

  • To hover on element and click
  • To enter uppercase C
Similarly, we can perform many other complex user interactions by chaining more actions

For more info, please refer: https://www.selenium.dev/selenium/docs/api/py/webdriver/selenium.webdriver.common.action_chains.html#selenium.webdriver.common.action_chains.ActionChains.key_down
